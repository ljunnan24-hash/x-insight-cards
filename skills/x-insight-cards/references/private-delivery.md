# Private WeChat delivery

Use this step only when the user has explicitly configured delivery to their own WeChat File Transfer Assistant. It is private review delivery, not social publishing.

1. Confirm that WeChat is signed in and the visible conversation title is exactly `文件传输助手`.
2. Obtain any just-in-time send confirmation required by the computer-control environment. A saved preference does not override a mandatory confirmation.
3. Send each final PNG as an image message, not as a file attachment.
4. Send its matching Chinese caption as a separate plain-text message immediately after the image. Never send a TXT file.
5. Paste Chinese from the verified local source. Before sending, copy it back from the message box and compare the complete text, punctuation, numbers, English, and hashtags.
6. Stop without sending when the destination or pasted text cannot be verified. Do not guess, switch recipients, or retry blindly.
7. Record the delivered item count and mark only successfully delivered items `DELIVERED_FOR_REVIEW`.

Never open a Xiaohongshu, Douyin, or other publishing page during this step.
