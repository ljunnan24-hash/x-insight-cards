# Contributing

Thanks for helping make high-signal creator workflows safer and easier to reuse.

## Good contributions

- Better Simplified Chinese line breaking and punctuation QA.
- Font discovery for Linux and Windows.
- Accessibility improvements and contrast checks.
- Evidence-backed scoring changes with examples.
- Test cases for deduplication, rendering, and malformed input.
- Documentation that makes installation shorter or clearer.

## Pull requests

1. Keep third-party posts, avatars, screenshots, credentials, and personal paths out of the repository. The only exception is project-owner-approved result evidence placed in `assets/proof/` and documented in `NOTICE.md`.
2. Use synthetic content in fixtures and examples.
3. Run `make check`.
4. Explain the user benefit and any behavior change.
5. Keep automatic publishing outside this project.
